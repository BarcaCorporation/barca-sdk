## ANSI CODE
<br/>
Set code identifiers in strings to get Unicode, UTF-8/16 results.

Example to enable ANSI code in your executable:

```sh
chcp 65001
```

<br/>

Pronto, UTF-8("65001") está ativado!

<br/>
<br/>
Consulte a lista completa de códigos:
